Usage: ./q2demo [args] <demofilename> [demofilename...]
Args:
  -c (output configstrings)
  -f (output frame number)
  -l (output layouts)
  -h (this help message)
  -j (output in JSON format
  -p (output only server print message (chat, obituaries, etc)
  -v (output verbose parsing information - each message parsed)
  -x <framestart:frameend> (crop a demo to frames xx:yy)
